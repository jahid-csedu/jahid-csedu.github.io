NewzGator

Duration: Jul 2025 – Present

NewzGator is a personal project where I designed and implemented the entire backend system for a news aggregation platform focusing on Bangladeshi newspapers. The system ingests, cleans, classifies, and clusters thousands of articles daily, delivering structured news data for further use by the frontend and other services.

The project emphasizes real-time content updates, ensuring readers receive fresh news directly from verified Bangladeshi news sources. A key technical challenge was designing the system to efficiently crawl, process, and redirect news content while maintaining SEO-friendly URLs and ensuring compliance with Google News indexing standards.

Key backend contributions include:

  • Automated Crawling & Scraping: Built custom crawlers to fetch articles from multiple Bangladeshi news sources with fault tolerance and retry mechanisms.
  • Data Cleaning & Deduplication: Implemented pipelines to normalize content, remove redundant/unused data, and handle duplicate articles efficiently.
  • News Classification (NLP): Integrated a fine-tuned BERT model to automatically categorize news into 15+ segments such as Politics, Business, Sports, and Technology.
  • News Clustering: Developed a distance matrix–based algorithm to group similar articles across different sources, improving information diversity.
  • Scalable Serverless Architecture: Deployed backend workflows using AWS Lambda, S3, SQS, and DynamoDB to handle ingestion and processing with minimal operational overhead.
  • High Throughput Processing: Optimized batch jobs and asynchronous pipelines (Spring Batch + Kafka) to process thousands of articles daily.

This project demonstrates my ability to design and implement end-to-end backend pipelines, apply machine learning for real-world classification tasks, and leverage cloud-native serverless architectures for scalability.

The platform is currently live at https://www.newzgator.com.

Technologies used: Python, BERT Classifiers and Transformers, AWS Lambda, S3, DynamoDB, CloudWatch, AWS API Gateway, Google Search Console
